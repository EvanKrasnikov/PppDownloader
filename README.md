# PppDownloader
A web application for retrieving PPP GNSS data

## How to use
Type in a terminal:
```bash
git clone https://github.com/EvanKrasnikov/PppDownloader.git
./mvnw clean install
```
Web service will be available at **_localhost:8080_**
